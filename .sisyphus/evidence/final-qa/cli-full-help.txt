Bittensor CLI (Rust)

Usage: btcli-rs [OPTIONS] <COMMAND>

Commands:
  wallet     Wallet management commands
  stake      Stake management commands
  transfer   Transfer commands
  register   Registration commands (POW, burned, root)
  root       Root network commands
  subnet     Subnet commands
  delegate   Delegate commands
  weights    Weights commands (set-weights, get-weights)
  metagraph  Metagraph commands (show, sync)
  help       Print this message or the help of the given subcommand(s)

Options:
      --network <NETWORK>          Network to connect to: finney, test, local, archive, latent-lite
      --wallet.name <WALLET_NAME>  Wallet name (defaults to "default")
      --wallet.path <WALLET_PATH>  Wallet base path (defaults to ~/.bittensor/wallets/)
  -h, --help                       Print help
  -V, --version                    Print version
Wallet management commands

Usage: btcli-rs wallet [OPTIONS] <COMMAND>

Commands:
  create            Generate new coldkey (encrypted) + hotkey pair
  list              List all wallets in the wallet directory
  show              Display wallet details (address, balance, hotkeys)
  balance           Show balance for a wallet/coldkey
  overview          Comprehensive wallet overview (stakes, delegations, etc.)
  transfer          Transfer TAO to another address
  swap-coldkey      Initiate coldkey swap process
  inspect           Show all keys and addresses in the wallet
  regen-coldkey     Regenerate coldkey from mnemonic
  regen-coldkeypub  Regenerate coldkeypub from public key
  create-hotkey     Create a new hotkey under a wallet
  regen-hotkey      Regenerate hotkey from mnemonic
  help              Print this message or the help of the given subcommand(s)

Options:
      --network <NETWORK>          Network to connect to: finney, test, local, archive, latent-lite
      --wallet.name <WALLET_NAME>  Wallet name (defaults to "default")
      --wallet.path <WALLET_PATH>  Wallet base path (defaults to ~/.bittensor/wallets/)
  -h, --help                       Print help
Stake management commands

Usage: btcli-rs stake [OPTIONS] <COMMAND>

Commands:
  add             Stake TAO to a hotkey
  remove          Unstake TAO from a hotkey
  move            Move stake between hotkeys
  swap            Swap stake between hotkeys
  list            List all stakes for a wallet
  get-stake       Query stake for a specific hotkey
  set-auto-stake  Enable or disable auto-staking for a hotkey
  help            Print this message or the help of the given subcommand(s)

Options:
      --network <NETWORK>          Network to connect to: finney, test, local, archive, latent-lite
      --wallet.name <WALLET_NAME>  Wallet name (defaults to "default")
      --wallet.path <WALLET_PATH>  Wallet base path (defaults to ~/.bittensor/wallets/)
  -h, --help                       Print help
